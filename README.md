# auth-form
Tiny app practicing using Firebase Authentication.
Client-side built with Vue 3

### Vue 3: https://vuejs.org/

### Vuex: https://vuex.vuejs.org/

### vue-router: https://router.vuejs.org/

### Firebase: https://firebase.google.com/docs

yarn install
